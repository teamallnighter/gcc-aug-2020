## Set Up 

### Download

```bash
git clone https://github.com/teamallnighter/gcc-aug-2020.git
```

### Install

```bash
npm install
```

### Commands

#### Local Server

```bash
npm run dev
```

#### Build

```bash
npm run build
```

#### Deploy

```bash
npm run deploy
```

#### Build and Deploy

```bash
npm run build-deploy
```
